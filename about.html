The Project

What is NIT KKR Search Engine?

NIT KKR Search Engine is an AI-powered academic platform built specifically for the students, teachers, and administration of National Institute of Technology, Kurukshetra. It centralises all academic resources — Previous Year Question papers, study material, and AI-assisted learning — into one intelligent, fast, and beautiful interface.

Students get free, instant access to PYQs filtered by branch, semester, subject, year, and exam type. Each question comes with a one-click AI tutor powered by Groq's Llama 3.3 model. Teachers upload PDFs and the platform automatically extracts questions, making them searchable and AI-ready in seconds.

Journey

Development Timeline

Phase 1 — Foundation
Core PYQ Search Engine
Built the student dashboard with branch/semester/subject filters, localStorage-based PYQ database, and basic PDF viewer.
Phase 2 — AI Integration
Ask AI per Question
Integrated Groq API (Llama 3.3-70b) with per-question context. Teachers can upload PDFs and questions are auto-extracted using PDF.js.
Phase 3 — Productivity Tools
Bookmarks, To-Do, Notifications
Added localStorage-powered bookmarks, smart to-do list with priority/filter, and real-time notifications when new papers are uploaded.
Phase 4 — Coming Soon
Study Groups, Analytics, Mobile App
Real-time collaboration, department-wide analytics dashboard, PWA support, plagiarism checker, and native mobile apps.
